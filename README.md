## Commands

Run front end (client)

- npm run dev

Run back end (server)

- npm start (production)
- npm run devStart (development)

Enter psql final as labber

- psql -U labber -d final

or

- psql
- \c final labber

Seed db (in server!)

- npm run db:reset
